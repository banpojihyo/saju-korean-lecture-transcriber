# Direct Session Rewrite Prompt

## 목적
- 자동 생성된 통합 학습 패키지 초안 `md`를 GPT-5.4 direct session에서 최종본으로 다시 쓸 때 사용할 공통 프롬프트를 정리한다.
- 입력은 보통 아래 3가지다.
  - 교정된 원문 `data/daglo/corr/script/**/*.script.txt`
  - 자동 초안 `data/summaries/**/md/**/*.md`
  - 재작성 메타데이터 `data/summaries/**/rewrite_manifest/*.jsonl`

## 권장 입력 순서
1. 원문 `script.txt`
2. 자동 초안 `md`
3. 같은 파일의 `rewrite_manifest` 항목

## 권장 프롬프트
```text
다음 학습 패키지 초안은 자동 생성본이므로 그대로 다듬지 말고, 원문을 다시 읽어 최종본으로 재작성해줘.

입력:
1. 교정된 script 원문
2. 자동 생성 md 초안
3. rewrite_manifest 메타데이터

요구사항:
- 출력은 한국어로 작성
- 제목은 그대로 유지
- 5개 섹션 구조 유지:
  1) 핵심 주제별로 나눠서 정리
  2) 핵심 개념 맵
  3) 예상 시험문제
  4) 시험 포인트와 실전 주의사항
  5) 꼭 공부해야 할 내용
- 자동 초안 문장을 단순 치환하지 말고 파일 전체를 다시 쓸 것
- 인접 시리즈 파일과 구분되는 포인트를 반드시 드러낼 것
- 예상 시험문제는 파일별로 새로 작성하고, 같은 질문 틀을 반복하지 말 것
- 시험 포인트와 실전 주의사항은 범용 문구를 반복하지 말고, 이 파일의 오해 포인트를 직접 적을 것
- 원문에 남은 구어체/잔재 표현은 학습용 문장으로 정리할 것
- 원문에 없는 사실은 추가하지 말 것
```

## 체크리스트
- 질문 5개가 모두 같은 골격인가?
- 시험 포인트가 다른 파일에도 그대로 붙여 넣을 수 있는 문장인가?
- 원문에서 가장 중요한 개념 2~3개가 실제로 설명형 문장으로 풀렸는가?
- 인접 파일과 차별점이 한 줄 이상 드러나는가?
- 자동 초안에 있던 어색한 빈도 키워드가 그대로 남아 있지 않은가?
